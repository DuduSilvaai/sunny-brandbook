# Task: Brief Creator

> **Owner:** @nova
> **Frameworks:** DSL Revolution (Haynes #1), 3-Act Structure (Tom Breeze)
> **Inputs:** hook(s) escolhido(s), business-context, formato/canal alvo

## Quando Executar

Após `*hooks` selecionar 1-3 vencedores. Antes de qualquer produção criativa (designer, editor, creator UGC).

## Output: Brief Production-Ready

```
🎬 CREATIVE BRIEF — {nome do criativo}
═══════════════════════════════════════════

METADATA:
- Campanha: {nome}
- Formato: [ ] Vídeo  [ ] Imagem estática  [ ] Carrossel  [ ] Reel  [ ] Stories
- Canal: [ ] Meta Feed  [ ] Reels  [ ] Stories  [ ] YouTube  [ ] TikTok
- Aspect ratio: {1:1 | 4:5 | 9:16 | 16:9}
- Duração (vídeo): {seg}
- Versão: v1.0

CONCEITO CENTRAL (1 frase):
{descrição do "big idea"}

HOOK (3 versões):
  v1: "{hook escolhido}"
  v2: "{variação a/b}"
  v3: "{variação c}"

ESTRUTURA DSL (timeline):
  0-3s    DISRUPT
    Visual: {descrição}
    Audio: {descrição}
    Texto on-screen: "{texto}"

  3-15s   SOLVE
    Visual: {descrição}
    Audio: {descrição/copy spoken}
    Texto on-screen: "{texto}"

  15s+    LEAD
    Visual: {descrição}
    Audio: {descrição}
    Texto on-screen: "{texto}"
    CTA: "{call to action específico}"

VISUAL REFERENCES:
  - {URL ou descrição de referência 1}
  - {URL ou descrição 2}
  - {URL ou descrição 3}

TOM DE VOZ:
  - {3-5 adjetivos: ex: "direto, irreverente, com humor BR"}
  - Pessoa: [ ] 1ª  [ ] 2ª  [ ] 3ª
  - Formal/informal: {nível 1-5}

CTA ESPECÍFICO:
  Texto botão: "{ex: 'Quero meu acesso' / 'Garantir vaga'}"
  Action: {Sign up | Buy | Learn more | Watch}
  URL: {link}

ELEMENTOS OBRIGATÓRIOS:
  [ ] Logo da marca (canto, primeiros 5s)
  [ ] CTA visual nos últimos 3s
  [ ] Disclaimer legal (se aplicável)
  [ ] Música/sound design
  [ ] Legendas (sempre — 85% Meta sem som)

ELEMENTOS A EVITAR:
  - {restrições brand safety}
  - {palavras proibidas Meta/Google}
  - {claims ilegais}

SPECS TÉCNICOS:
  - Resolution: {min 1080x1080 ou 1080x1920}
  - File format: MP4 (vídeo) / PNG ou JPG (estático)
  - Max file size: 4GB (Meta)
  - Frame rate (vídeo): 30fps min
  - Audio bitrate: 128kbps min

PROVA / CASES (incluir se possível):
  - {nome + resultado}
  - {nome + resultado}

VARIATIONS REQUEST (Brandon Carter #1 — Constants vs Variables):
  Mesmo brief, mas trocar APENAS:
    [ ] Hook (3 versões já listadas)
    [ ] Demonstração (mostrar produto vs mostrar resultado)
    [ ] CTA wording

DEADLINE: {data}
DELIVERABLES:
  [ ] Master file (highest quality)
  [ ] 1:1 cut
  [ ] 4:5 cut
  [ ] 9:16 cut
  [ ] Thumbnail/cover (vídeo)

OWNER: {designer/editor responsável}
APPROVAL: @nova → @midas → go-live
```

## Constraints

- NUNCA mande brief sem hook validado
- NUNCA mande brief sem CTA específico (não "saiba mais")
- SEMPRE 3 versões de hook (variations sem mudar conceito)
- SEMPRE inclua legendas (85% Meta consumido sem som)
- SEMPRE specs técnicos para canal alvo
- NUNCA permita brief vago — designer precisa execução clara

# Vocabulário Oficial — Traffic Masters Squad

> **Regra:** Todos os agentes devem usar EXATAMENTE estes termos. Português + termos técnicos em inglês quando consagrados.

## Métricas

| Termo | Definição | NÃO usar |
|-------|-----------|----------|
| **ROAS** | Return on Ad Spend = Revenue / Spend | "ROI" (genérico, ambíguo) |
| **CAC** | Custo de Aquisição de Cliente | "custo por cliente" |
| **nCAC** | new Customer Acquisition Cost (só novos clientes via paid) | misturar com CAC blended |
| **LTV** | Lifetime Value | "valor do cliente" |
| **CPA** | Cost Per Acquisition (por evento de conversão) | "custo por venda" |
| **CPM** | Cost Per Mille (1.000 impressões) | "custo por mil" |
| **CPC** | Cost Per Click (link click) | "custo por clique" |
| **CTR** | Click-Through Rate (link click / impressões) | "taxa de clique" |
| **AOV** | Average Order Value | "ticket médio" (ok mas prefira AOV) |
| **EMQ** | Event Match Quality (Meta CAPI score 0-10) | "qualidade do pixel" |
| **Hook Rate** | % de pessoas que assistem 3s de vídeo | "retenção inicial" |
| **Hold Rate** | % de pessoas que assistem > 25% do vídeo | "retenção" |

## Estrutura de Campanha

| Termo | Definição |
|-------|-----------|
| **CBO** | Campaign Budget Optimization (budget na campanha) |
| **ABO** | Ad Set Budget Optimization (budget no ad set) |
| **Ad Set** | Conjunto de anúncios com mesma audiência/budget |
| **Creative** | Anúncio individual (criativo + copy) |
| **Placement** | Onde o ad aparece (Feed, Stories, Reels, etc.) |
| **Learning Phase** | Fase de aprendizado (50 conversões/semana para sair) |
| **Creative Fatigue** | Fadiga criativa (queda de performance por saturação) |

## Audiências

| Termo | Definição |
|-------|-----------|
| **LAL** | Lookalike Audience (1%, 3%, 5% baseado em seed) |
| **Custom Audience** | Audiência customizada (pixel, email list, engagement) |
| **Broad** | Sem targeting (só geo + idade) |
| **Interest Stack** | 3-5 interesses correlatos no mesmo ad set |
| **Retargeting** | Audiência que já interagiu (warm/hot) |
| **Cold Traffic** | Audiência que nunca interagiu |

## Funil

| Termo | Definição |
|-------|-----------|
| **TOFU** | Top of Funnel (consciousness) |
| **MOFU** | Middle of Funnel (consideration) |
| **BOFU** | Bottom of Funnel (decision) |
| **VSL** | Video Sales Letter |
| **Tripwire** | Oferta low-ticket que cobre CAC (Hormozi R$17) |
| **Order Bump** | Adicional no checkout |
| **Upsell** | Oferta adicional pós-compra |
| **Lead Magnet** | Conteúdo gratuito em troca de email |

## Operações

| Termo | Definição |
|-------|-----------|
| **Kill Rule** | Regra automática de pausar |
| **Scale Rule** | Regra automática de aumentar budget |
| **Vertical Scale** | Aumentar budget na mesma estrutura |
| **Horizontal Scale** | Duplicar estrutura com nova audiência |
| **Refresh** | Trocar criativos para combater fadiga |
| **Iteration** | Variação de criativo existente |

## Tracking

| Termo | Definição |
|-------|-----------|
| **CAPI** | Conversions API (server-side, pós-iOS14) |
| **Pixel** | Tag client-side de tracking |
| **Eventos Padrão** | PageView, ViewContent, AddToCart, InitiateCheckout, Purchase, Lead |
| **Eventos Customizados** | Eventos definidos para o negócio |
| **Atribuição** | Modelo de crédito por canal/touchpoint |
| **Match Rate** | % de eventos com user match no Meta |

## Decisões

| Termo | Definição |
|-------|-----------|
| **Auto-decision** | Decisão tomada sem interação humana (com log) |
| **Dispatch** | MIDAS delegando para especialista |
| **Handoff** | Transição entre agentes com contexto |
| **Quality Gate** | Checkpoint obrigatório antes de avançar |

---

**Regra final:** Quando em dúvida entre português e inglês, prefira o termo técnico EM INGLÊS se for jargão consagrado da indústria (ROAS, CAC, CBO, LAL). Use português para verbos e contexto ("vamos pausar o ad set", "aplicar kill rule", "expandir audiência").

# Task: Angle Generator

> **Owner:** @nova
> **Frameworks:** 10+ Ângulos (Haynes implícito) + Schwartz Awareness Levels + Brandon Carter Variation
> **Inputs:** business-context, oferta, persona

## O que é "Ângulo"

Ângulo ≠ Hook. Ângulo é a ABORDAGEM CONCEITUAL completa — a tese central do criativo. Hook é a primeira frase de UM ângulo.

```
ÂNGULO: "Fear of missing out: Black Friday agora ou daqui 1 ano"
HOOKS desse ângulo:
  - "Última chance esse ano de [oferta]"
  - "Vai esperar 365 dias pra próxima Black Friday?"
  - "Black Friday 2026 acaba em 12 horas"
```

## 10 Ângulos Universais

### 1. PAIN ANGLE
Foca na dor atual do avatar.
```
"Sua campanha está sangrando R$ e você não sabe parar"
```

### 2. RESULT ANGLE
Foca no resultado final desejado.
```
"R$50k em 30 dias — método replicável"
```

### 3. COMPARISON ANGLE
Vs concorrente, vs status quo, vs alternativa.
```
"Esqueça [concorrente]. Aqui está o que funciona em 2026"
"Pare de pagar agência R$15k. Faça você mesmo."
```

### 4. AUTHORITY ANGLE
Construir credibilidade do mensageiro.
```
"Eu gerenciei R$50M em ads. Isso é o que aprendi."
```

### 5. FEAR / FOMO ANGLE
Medo de perder, perigo, urgência.
```
"Black Friday acaba em 12h" / "Algoritmo Meta vai mudar em outubro — adapta agora"
```

### 6. ASPIRATIONAL ANGLE
Vida ideal, identidade aspiracional.
```
"Como é ser dono do próprio tempo trabalhando 4h/dia"
```

### 7. EDUCATIONAL ANGLE
Ensina algo (e vende como conclusão).
```
"3 erros que matam suas campanhas no D7 (e como evitar)"
```

### 8. ORIGIN STORY ANGLE
História pessoal de transformação.
```
"Em 2020 eu estava falido. Hoje gerencio R$2M/mês em ads. Aqui está como."
```

### 9. BEHIND-THE-SCENES ANGLE
Mostra processo, autenticidade.
```
"Como eu organizo meu ad account de R$500k/mês (vídeo de tela)"
```

### 10. CONTRARIAN ANGLE
Vai contra o consenso da indústria.
```
"Todo mundo fala em CBO. Eu uso ABO e tenho 3x mais ROAS. Aqui está por quê."
```

## Awareness Match (Schwartz)

```
AWARENESS LEVEL              → ÂNGULOS QUE FUNCIONAM
─────────────────────────────────────────────────
Inconsciente                  → Educational, Origin, Behind-the-scenes
Consciente do problema        → Pain, Fear/FOMO
Consciente da solução         → Comparison, Educational
Consciente do produto         → Authority, Result
Mais consciente               → Contrarian, Aspirational, Comparison
```

## The 4-Question Triage

Quando você chega sem saber o que precisa, eu rodo ESSE triage — sempre. Não é opcional.

**Pergunta 1: Qual é o produto e o ticket médio?**
- Sem isso eu não consigo escolher o tipo de funil. Preço dita estrutura.

**Pergunta 2: Quem é o cliente ideal e onde ele está?**
- Sem avatar, Dream 100 é impossível. Sem Dream 100, tráfego é loteria.

**Pergunta 3: O que está quebrando agora?**
- Sem tráfego? Sem conversão? Sem oferta clara? Sem história?
- Essa resposta escolhe o livro da trilogia que resolve.

**Pergunta 4: Qual é a meta dos próximos 90 dias?**
- Mais leads? Mais receita? Primeira venda? Scale?
- Meta dita o funil que vamos construir.

Com as 4 respostas eu te entrego: **o próximo passo exato, com dono e prazo.**

## Decision Trees (Determinísticas)

### Árvore 1 — Tipo de Funil (por ticket médio)

```
SE ticket < R$50         → Squeeze Page + Tripwire (OTO stacking)
SE ticket R$50-R$500     → Sales Letter / VSL
SE ticket R$500-R$2.000  → Webinar Funnel (Perfect Webinar)
SE ticket R$2.000-R$10k  → Application Funnel (webinar + call)
SE ticket > R$10.000     → High-Ticket Phone Funnel (2-call close)
SE produto é SaaS        → Free Trial + Membership Funnel
SE é Network Marketing   → Bridge Page + 1 dos 3 Lost Funnels
SE é info-produto nicho  → Product Launch Funnel (4 vídeos, 7-10 dias)
```

### Árvore 2 — Problema de Conversão (diagnóstico ordenado)

```
SE opt-in < 25%               → Problema na HEADLINE ou no BAIT (lead magnet errado)
SE opt-in OK, vendas = 0      → Problema na OFERTA ou na CRENÇA (Big Domino)
SE vendas OK, OTO < 10%       → OTO não complementa o core (ou tem preço errado)
SE webinar show-up < 20%      → Problema no email de confirmação/reminder
SE webinar fechamento < 5%    → Script fraco — Big Domino mal definida
SE cart abandonment > 70%     → Problema de price anchor (sem stack slide)
SE refund > 10%               → Expectation mismatch (oferta prometeu além do entregue)
SE LTV/CAC < 2                → Value Ladder incompleto (falta continuity ou backend)
```

### Árvore 3 — Ordem de Investigação em Funil Quebrado

```
1. TRAFFIC TEMPERATURE   → tráfego frio indo pra página quente?
2. OFFER STRENGTH        → oferta é fraca? sem stack, sem urgência, sem scarcity?
3. HOOK                  → está furando o feed? parando o scroll?
4. STORY                 → derruba as 3 crenças falsas?
5. FUNNEL STRUCTURE      → o tipo de funil é o certo pro objetivo?
6. COPY                  → por último. Copy é sintoma, não causa.
```

Quem começa diagnóstico por "copy" está perdendo tempo.

### Árvore 4 — Traffic Temperature Matching

```
FRIO (não sabe quem você é)     → Bridge page educativa, história primeiro, CTA soft
MORNO (já ouviu falar)          → Case study, social proof, webinar
QUENTE (já comprou / na lista)  → Oferta direta, urgência, OTO
```

Mandar tráfego frio pra página quente = suicídio de conversão. Match sempre.

## Thresholds (Kill / OK / Scale)

Benchmarks operacionais — quando matar, quando manter, quando escalar. (Referência: chunk-09-thresholds.md)

### Opt-in Page (Squeeze)

| Métrica | Kill | OK | Scale |
|---------|------|-----|-------|
| Opt-in rate (cold traffic) | < 15% | 20-30% | > 35% |
| Opt-in rate (warm traffic) | < 25% | 30-45% | > 50% |
| CPL (cold) | > R$20 | R$5-R$15 | < R$5 |

### Webinar Funnel

| Métrica | Kill | OK | Scale |
|---------|------|-----|-------|
| Registration rate | < 30% | 40-55% | > 60% |
| Show-up rate | < 20% | 25-40% | > 45% |
| Close rate (ao vivo) | < 3% | 5-10% | > 12% |
| Close rate (replay) | < 2% | 3-7% | > 8% |
| EPC (Earnings Per Click) | < R$2 | R$5-R$15 | > R$20 |

### Tripwire / Cart Funnel

| Métrica | Kill | OK | Scale |
|---------|------|-----|-------|
| Frontend conversion | < 5% | 8-15% | > 18% |
| OTO1 take rate | < 20% | 30-45% | > 50% |
| OTO2 take rate | < 10% | 15-25% | > 30% |
| AOV | < 1.5x frontend | 2-3x | > 3.5x |
| ROAS (cold) | < 1.5 | 2-3 | > 3.5 |

### Challenge Funnel

| Métrica | Kill | OK | Scale |
|---------|------|-----|-------|
| Opt-in rate | < 30% | 40-55% | > 60% |
| Day 1 → Day 5 retention | < 25% | 40-55% | > 60% |
| Day 5 close rate | < 5% | 8-15% | > 18% |

### Economia da Value Ladder

| Métrica | Kill | OK | Scale |
|---------|------|-----|-------|
| LTV/CAC | < 2 | 3-5 | > 5 |
| Payback period | > 90d | 30-60d | < 30d |
| Stack perceived value / price | < 5x | 8-15x | > 20x |

**Regra:** não escala o que não bate thresholds de OK por 14 dias consecutivos. Escalar KILL é jogar dinheiro fora.

## Core Philosophy (5 Princípios Inegociáveis)

### 1. Todo negócio TEM um funil — a questão é se é intencional ou não
Seu concorrente tem website. Você tem funil. Essa é a diferença entre quebrar e escalar.

### 2. O único problema real é falta de clientes
E isso se resolve com DUAS coisas: funil + tráfego. Ponto. Tudo o mais é desculpa.

### 3. Você não vende produtos, vende TRANSFORMAÇÃO
Ninguém compra broca — compra buraco na parede. Ninguém compra curso — compra o STATUS que tem depois de fazer o curso.

### 4. Teste antes de escalar — NUNCA escale o que não provou
O gráfico não mente. Se não bate thresholds de OK em 14 dias, mata.

### 5. A lista é o ativo mais valioso
Algoritmo muda, plataforma morre, ads ficam caros. Lista é sua. Email entrega. O dinheiro está no follow-up — Soap Opera + Seinfeld.

**Bonus:** Hook. Story. Offer. É física. Não é opinião.

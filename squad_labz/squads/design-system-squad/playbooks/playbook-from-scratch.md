# Playbook: Build Design System from Scratch

For projects **starting a design system from zero**.

**Phases:** 0 → 1 → 2 → 3 → 4 → 5 → 6

## Phase 0: Business Discovery
**Agent:** @design-architect
**Duration:** 1-2 days
**Output:** `business-context.md`

- Define design system purpose
- Identify stakeholders
- Set success metrics
- Align on vision

## Phase 1: Foundation
**Agents:** @alla-kholmatova + @brad-frost
**Duration:** 3-5 days
**Output:** Purpose statement + Atomic architecture design

- Establish shared language
- Define design philosophy
- Design component hierarchy (Atomic Design)

## Phase 2: Strategy
**Agents:** @dan-mall + @dave-malouf + @heydon-pickering
**Duration:** 5-7 days
**Output:** Business case + DesignOps plan + Accessibility standards

- Build ROI business case
- Design DesignOps processes
- Define accessibility requirements

## Phase 3: Building
**Agents:** @brad-frost + @jina-anne + @sarah-drasner + @una-kravets
**Duration:** 10-14 days
**Output:** Component library + Tokens + Motion system + CSS architecture

- Build initial atomic components
- Extract design tokens
- Design motion patterns
- Build CSS system

## Phase 4: Governance
**Agent:** @nathan-curtis
**Duration:** 2-3 days
**Output:** Governance model + Versioning strategy + Contribution guidelines

- Define governance structure
- Plan versioning approach
- Create contribution process

## Phase 5: Quality & Accessibility
**Agents:** @heydon-pickering + @ux-design-expert
**Duration:** 3-5 days
**Output:** Accessibility audit + User research findings

- Full WCAG 2.2 audit
- User testing with initial audience
- Remediation planning

## Phase 6: Launch
**Agent:** @design-architect (coordinates all)
**Duration:** 2-3 days
**Output:** Launched design system + Documentation + Adoption plan

- Final review and approval
- Team onboarding
- Launch to organization

---

**Total Duration:** ~30-40 days of focused work
**Team Size:** 2-3 people minimum (plus agent orchestration)
**Success Criteria:** All quality gates passing, business context clear, accessibility AAA

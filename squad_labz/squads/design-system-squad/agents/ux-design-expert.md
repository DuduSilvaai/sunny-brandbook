# UX Design Expert - External Integration

## About This Agent

This is an **external reference** to the UX Design Expert agent (Uma) defined in `.aios-core/development/agents/ux-design-expert.md`.

Uma specializes in **user research, usability testing, and user validation** for design systems.

## When to Route to Uma

Route when:
- Need user research for design system decisions
- Validating design system with real users
- Conducting usability testing
- Gathering user feedback on components
- Making evidence-based design decisions

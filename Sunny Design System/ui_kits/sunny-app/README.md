# Sunny App UI Kit

## Overview
Interactive prototype for the **Sunny OS** SaaS application — the "Torre de Comando" for solar sales intelligence.

## Screens
1. **Dashboard (Torre de Comando)** — KPI metrics, pipeline overview, activity feed
2. **Pipeline (Sunny Pipeline)** — Kanban CRM with lead cards and stage management
3. **SDR Agent (Sunny Agent)** — AI conversation monitor with live leads
4. **Lead Detail** — Full lead profile with conversation history

## Usage
Open `index.html` to view the interactive prototype. Use the sidebar to navigate between screens.

## Components
- `Sidebar.jsx` — App navigation sidebar
- `Dashboard.jsx` — Torre de Comando dashboard
- `Pipeline.jsx` — Kanban pipeline view
- `AgentView.jsx` — SDR Agent conversation view
- `LeadCard.jsx` — Reusable lead card component

## Design Specs
- Design width: 1280px
- Dark mode only
- Font: Space Grotesk (brand) + Inter (UI) via Google Fonts
- Color tokens from `../../colors_and_type.css`

# Sunny — Referência Visual

## Direção Geral
- **Estilo:** Dark Premium / Luxury Tech
- **Modo:** Dark mode como padrão (fundos pretos profundos)
- **Accent:** Cores solares vibrantes (a definir — NÃO precisa ser laranja)
- **Tipografia:** Space Grotesk (geométrica, moderna)
- **Qualidade:** Magazine-quality, editorial, sofisticado

## Logo Concept (Da Referência)
- **Logomark:** Sol estilizado com forma de "S" integrada + raio ⚡
- **Composição:** "S" (letra) + elemento solar/energético
- **Estilo:** Ícone limpo, geométrico, escalável
- **Variações necessárias:** Full lockup (ícone + wordmark), ícone only, monochrome, reversed

## Tipografia
- **Brand typeface:** Space Grotesk
  - Pesos: Thin, Regular, Bold, Black
  - Geométrica com personalidade
  - Moderna e legível
- **Considerar também:** 
  - Display: variante para títulos grandes
  - Mono: para dados/código (dashboard context)

## Paleta de Cores (Em Aberto)
**Direção do PRD:** "Clean Luxury Tech / Elegant Dark Mode"

Sugestões de direcionamento:
- **Fundos:** Preto profundo (#0A0A0A a #111111)
- **Accent solar:** Pode ser dourado, âmbar, coral solar, ou gradiente warm
- **Superfícies:** Cinzas escuros com glassmorphism sutil
- **Branco:** Warm white para texto

> NOTA: As cores finais serão definidas por michael-johnson na Fase 4, considerando o arquétipo e posicionamento das Fases 1-2.

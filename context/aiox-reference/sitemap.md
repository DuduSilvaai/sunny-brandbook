## Elementos-Chave do AIOX Brandbook

### Identity System
- **Typeface primária:** Geist (todas as aplicações)
- **Display:** Tasa Orbiter 800
- **Mono:** Roboto Mono 500
- **Cor primária:** Kinetic Limon #D1FF00
- **Tema:** Dark Cockpit (fundo escuro, accent neon)
- **Design Tokens:** CSS variables (--BB-DARK, --BB-SURFACE, --BB-WARM-WHITE)

### Manifesto
> "EU NÃO PRECISO SER PROGRAMADOR PARA CRIAR. A IA É A SETA. O X É MEU."

### Símbolos
- **∆ Triângulo/Delta** — Transformação, jornada
- **🕹️ Joystick** — Controle do criador
- **→ Seta** — IA como meio propulsor
- **X** — "X Marks the Spot", o destino

### Naming Semântico: A → I → O → X
- A = A Seta (ponto de partida)
- I = Input (histórias do usuário)
- O = Orquestração (pipelines)
- X = O Destino (resultado final)

### Archetypes
- **Magician 60%** — Transformação
- **Sage 25%** — Sabedoria, método
- **Explorer 15%** — Autonomia, aventura
- Analogia: **Morpheus** (Matrix)

### Dual Voice
- **AIOX (Produto):** Frio, implacável, minimalista, sábio
- **Alan (Fundador):** Quente, passional, expansivo, provocador

### Vocabulário
- **Power Words:** O X, A Seta, O Terminal, A Clareza, Transformador, Revelador, Direto, Jornada
- **Ban List:** Mágico, Revolucionário, Fácil, Hack

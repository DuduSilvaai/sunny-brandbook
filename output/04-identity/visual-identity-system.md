## SECTION 1: BRAND STORY

### Purpose
A Sunny existe para **eliminar a morte comercial** no setor de energia solar. Cada lead que não é respondido é uma família que não economiza com energia solar, uma integradora que perde receita, e um planeta que esquenta um pouco mais.

### Values
1. **Intolerância à Inação** — Cada minuto sem resposta é uma venda morrendo
2. **Magnanimidade Operacional** — Grandioso onde o mercado aceita medíocre
3. **Clareza Solar** — Zero ruído, cada dado responde uma pergunta real
4. **Soberania do Integrador** — A IA amplifica, nunca substitui
5. **Velocidade com Fundação** — Rápido E robusto

### Personality
- **Comandante** — Fala com autoridade, transmite confiança
- **Magnânima** — Generosa com valor, experiência premium
- **Precisa** — Dados exatos, promessas concretas
- **Incansável** — 24/7, sem desculpas
- **Luminosa** — Clareza onde havia escuridão

### Archetype
**Ruler Magnânimo** (60%) + Magician (25%) + Caregiver (15%)

## SECTION 4: TYPOGRAPHY SYSTEM

### Brand Typeface: **Space Grotesk**

| Propriedade | Valor |
|-------------|-------|
| **Papel** | Headlines, marketing, hero sections, logo wordmark |
| **Categoria** | Sans-serif geométrica |
| **Fonte** | Google Fonts (open source, gratuita) |
| **Weights** | Medium (500), Bold (700) |
| **Rationale** | Geométrica com personalidade — não é fria como Helvetica nem decorativa como display fonts. Transmite comando + modernidade. O "r" e "g" minúsculos têm character. Funciona em PT-BR e EN sem problemas. |
| **Kerning** | Tight (-0.02em em headlines) — alinhado com estilo "kerning denso" do PRD |

### UI Typeface: **Inter**

| Propriedade | Valor |
|-------------|-------|
| **Papel** | Body text, UI de produto, dashboard, documentação |
| **Categoria** | Sans-serif humanista otimizada para telas |
| **Fonte** | Google Fonts (open source, gratuita) |
| **Weights** | Regular (400), Medium (500), SemiBold (600), Bold (700) |
| **Rationale** | Desenhada especificamente para interfaces digitais. Legibilidade perfeita em tamanhos pequenos. Pair natural com Space Grotesk — ambas são geométricas mas Inter é mais neutra, criando hierarquia visual clara. |

### Mono Typeface: **JetBrains Mono**

| Propriedade | Valor |
|-------------|-------|
| **Papel** | Código, dados tabulares, timestamps, IDs, métricas |
| **Categoria** | Monospace |
| **Fonte** | Google Fonts |
| **Weights** | Regular (400), Medium (500) |
| **Rationale** | Legibilidade excepcional em mono. Ligatures opcionais. Conecta com o aspecto técnico/dados da marca. |

## SECTION 5: LOGO SYSTEM

### Logo Concept (Inherited from Reference)
O logomark combina três elementos semânticos:

| Elemento | Forma | Significado |
|----------|-------|-------------|
| **S** | Letra S estilizada | Sunny, Solar, Speed |
| **☀️ Sol** | Raios/corona ao redor | Energia solar, iluminação, "nunca se apaga" |
| **⚡ Raio** | Integrado no S | Velocidade, IA, eletricidade, poder |

### Logo Lockups

| Variação | Composição | Uso |
|----------|-----------|-----|
| **Full Lockup** | Logomark + "Sunny" wordmark (horizontal) | Uso principal, marketing, header |
| **Stacked** | Logomark acima + "Sunny" abaixo | Quadrados, social profiles |
| **Icon Only** | Apenas logomark | Favicon, app icon, loading |
| **Wordmark Only** | Apenas "Sunny" em Space Grotesk Bold | Quando o ícone já está em contexto |

### Clear Space
- Mínimo: **1x a altura do ícone** em todos os lados
- Recomendado: **1.5x** para máximo impacto

### Minimum Size
- Digital: **24px** de altura (icon only), **32px** (full lockup)
- Print: **10mm** de altura (icon only), **15mm** (full lockup)

### Logo Colors

| Versão | Fundo | Logo | Uso |
|--------|-------|------|-----|
| **Primary** | Command Black | Solar Flare gradient | Uso padrão, dark mode |
| **Reversed** | Solar Flare | Command Black | Backgrounds claros, impressos solares |
| **Mono Light** | Command Black | Foreground (#FAFAFA) | Minimal, overlay em foto |
| **Mono Dark** | Light bg | Command Black | Documentos formais |

### Logo DON'Ts ❌
1. ❌ Nunca esticar ou distorcer
2. ❌ Nunca rotacionar
3. ❌ Nunca aplicar drop shadow ou outer glow artificial
4. ❌ Nunca colocar sobre fundo concorrendo (foto muito detalhada sem overlay)
5. ❌ Nunca mudar as proporções ícone/wordmark
6. ❌ Nunca usar cores fora da paleta
7. ❌ Nunca adicionar stroke ou outline
8. ❌ Nunca reduzir abaixo do mínimo
9. ❌ Nunca usar o ícone como pattern repetido

### Photography Style

| Dimensão | Diretriz |
|----------|---------|
| **Lighting** | Rim light, golden hour, contrastes dramáticos (Chiaroscuro tech) |
| **Color** | Predominantemente dark com pontos de luz solar/dourada |
| **Subjects** | Integradoras reais trabalhando, dashboards em uso, painéis solares em contexto urbano |
| **Composition** | Clean, assimétrica, espaço negativo generoso |
| **Mood** | Confiança, autoridade, competência — NÃO "happy stock photo" |
| **Avoid** | Fotos genéricas de handshake, painéis em campo verde sem contexto, pessoas apontando para telas |

### Image Treatment
- **Overlay:** Black com 50-70% opacity + gradient solar no edge
- **Duotone:** Command Black + Solar Flare para imagery editorial
- **Filter:** Contraste +15%, Shadows -10%, Warmth +5%

### Iconography
- **Style:** Line icons, stroke 1.5px, rounded caps
- **Grid:** 24x24px com safe area de 2px
- **Família:** Solar-themed quando possível (raio, sol, pipeline, alvo, calendário)
- **Cor:** Foreground (#FAFAFA) ou Solar Flare (#F5A524) em dark backgrounds

| Primary Text | #09090B |
| Secondary Text | #71717A |
| Accent | Dawn Amber (#E8740C) — ajustado para contraste em fundo claro |
| Borders | #E4E4E7 |

### Motion Guidelines

| Tipo | Duração | Easing | Uso |
|------|---------|--------|-----|
| **Micro** | 150ms | ease-out | Hover, focus, toggle |
| **Standard** | 250ms | ease-in-out | Opens, closes, expandos |
| **Emphasis** | 400ms | cubic-bezier(0.16, 1, 0.3, 1) | Page transitions, modals |
| **Dramatic** | 600-800ms | spring(1, 100, 10, 0) | Hero animations, logo reveal, onboarding |

**Princípios de Motion:**
1. Tudo se move com propósito — nenhuma animação decorativa
2. Elementos entram de baixo para cima (ascensão = sol nascendo)
3. CTAs pulsam sutilmente com Solar Glow em idle
4. Loading states usam skeleton gradients (Surface → Surface Hover)
5. Transições entre páginas: fade + slide up (200ms)

## SECTION 9: LOGO EVALUATION (SAGI HAVIV — 5 PILLARS)

### Avaliação do Logomark (Sol + S + Raio)

| Pilar | Score | Avaliação |
|-------|-------|-----------|
| **1. Appropriateness** | 8/10 | O logo é altamente apropriado para o setor (sol = solar) e para a marca (S = Sunny, raio = velocidade/IA). Conecta emocionalmente com o integrador que vende energia do sol. |
| **2. Distinctiveness** | 7/10 | Se diferencia dos CRMs genéricos (que usam ícones abstratos ou lettering). O combo S+sol+raio é único no setor solar brasileiro. Ponto de atenção: garantir que o raio não pareça "eletricidade genérica". |
| **3. Simplicity** | 7/10 | Três elementos (S, sol, raio) estão no limite do aceitável. Funciona em tamanho pequeno? Testar. Se necessário, simplificar removendo detalhes dos raios solares. Menos é mais. |
| **4. Memorability** | 8/10 | A combinação é memorável porque o significado é claro: sol+velocidade. Passa no "draw from memory test" — alguém poderia desenhar uma aproximação após ver uma vez. |
| **5. Timelessness** | 7/10 | Risco: raios solares podem parecer datados se forem muito detalhados. Recomendação: manter formas geométricas puras — arcos, não detalhes orgânicos. Geometria envelhece melhor. |

**Score Total: 37/50**  
**Veredicto:** O conceito de logo é **sólido**. Com refinamento na simplificação e na execução geométrica, pode atingir 42+/50.

### Recomendações do Sagi Haviv
1. **Simplificar** — Reduzir o número de raios solares ao mínimo expressivo
2. **Unificar** — O S, o sol e o raio devem parecer UMA forma, não três sobrepostas
3. **Testar em 16x16** — Se não funciona como favicon, simplificar mais
4. **Monochrome first** — Desenhar em preto e branco antes de aplicar cor
5. **Negative space** — Usar espaço negativo para criar profundidade (o raio pode ser negative space dentro do S)

## 1. CONTEXTO ESTRATÉGICO DA MARCA

### O que é o Sunny?
**Sunny** é uma plataforma SaaS de **Inteligência Comercial Solar** que usa IA (SDR Agents) para eliminar a inação comercial no setor de energia solar brasileiro. O produto automatiza qualificação, follow-up e agendamento de leads — respondendo em menos de 15 minutos, 24/7.

### Posicionamento (Onliness Statement)
> "Sunny é a **única** plataforma de inteligência comercial solar que transforma leads perdidos em receita previsível através de SDR Agents autônomos que nunca dormem."

### Arquétipo de Marca
| Primário | Secundário |
|----------|------------|
| **Ruler** (Controle, Autoridade, Ordem) | **Magician** (Transformação, Visão) |

> Referência Pantheon: **Tesla** (Ruler + Magician) — Power through transformation.
> Referência Pantheon: **Stripe** (Sage) — Infraestrutura invisível com DX premium.

### A Palavra que Sunny Deve Possuir
**COMANDO** — No sentido de "torre de comando da guerra comercial solar".

### O Shared Enemy
**A Inação Comercial** — leads que morrem sem resposta, CRM abandonado, pipeline com buracos.

### Tagline
> "Nenhum lead morre ao sol."

### Padrões Observados (O que TODOS fazem)
1. ☀️ **Sol literal** — raios simétricos irradiando do centro
2. 🔵 **Azul corporativo** — 6 de 9 usam azul como cor primária
3. 📝 **Wordmarks genéricos** — tipografia sem personalidade
4. 🏗️ **Estilo "solar residencial"** — visual de catálogo, não de software premium
5. 🤷 **Zero personalidade** — nenhum transmite comando, poder ou IA

### Oportunidade de Diferenciação
> **Quando todos zigam, zague.** (Marty Neumeier)

- ❌ NÃO usar azul corporativo
- ❌ NÃO usar sol com raios simétricos (clichê solar)
- ❌ NÃO parecer "painéis solares" ou "energia renovável"
- ✅ PARECER: Software de elite, IA, controle, autoridade
- ✅ TOM: Dark premium, luxury tech (como Stripe, Linear, Vercel)
- ✅ COR: Dourado âmbar flamejante sobre negro profundo — oposto completo do mercado

## 4. CONCEITO CRIATIVO — "O COMANDO SOLAR"

### 4.1 Narrativa do Símbolo

A logo da Sunny não é um sol. **É uma torre de comando energética.**

A metáfora visual combina três camadas de significado:

```
CAMADA 1 — ENERGIA SOLAR
   O sol como fonte universal de energia.
   Não raios simétricos — mas um halo de poder contido.

CAMADA 2 — VELOCIDADE / RELÂMPAGO
   A letra "S" formando um raio que corta o centro.
   Velocidade: <15 minutos. Ação instantânea.
   
CAMADA 3 — COMANDO / ESCUDO
   Uma forma geométrica que contenha, proteja e projete poder.
   Referência: brasões, moedas, selos de autoridade.
```

### 4.2 Anatomia do Logomark

```
┌─────────────────────────────────────┐
│                                     │
│    ┌──── CONTAINER EXTERNO ────┐    │
│    │   Círculo ou hexágono     │    │
│    │   (Escudo / Moeda / Selo) │    │
│    │                           │    │
│    │   ┌── ELEMENTO CENTRAL ─┐ │    │
│    │   │   Letra "S" em      │ │    │
│    │   │   formato de raio   │ │    │
│    │   │   (lightning bolt)   │ │    │
│    │   └─────────────────────┘ │    │
│    │                           │    │
│    │   ┌── HALO ENERGÉTICO ──┐ │    │
│    │   │   Arco ou semicírcl.│ │    │
│    │   │   representando o   │ │    │
│    │   │   horizonte solar   │ │    │
│    │   └─────────────────────┘ │    │
│    │                           │    │
│    └───────────────────────────┘    │
│                                     │
│    + WORDMARK: "Sunny"              │
│      Space Grotesk Bold             │
│      Kerning denso (-2%)            │
│                                     │
└─────────────────────────────────────┘
```

### 4.3 Decisões de Design Específicas

| Decisão | Escolha | Justificativa |
|---------|---------|---------------|
| **Container** | Círculo com borda dashed sutil | Selo de autoridade + "órbita solar". Referência: moedas, medalhas, selos de garantia |
| **Elemento Central** | Letra S em forma de bolt/raio | Fusão da inicial "S" de Sunny com o ⚡ de velocidade/energia. Gestalt dupla |
| **Proporção** | S ocupa ~60% do container | Protagonismo claro. O símbolo é o raio, não o círculo |
| **Ângulo do Raio** | 15° de inclinação para direita | Transmite movimento para frente, progressão, antecipação |
| **Halo** | Arco inferior a 180° | "Sol nascente" no horizonte — referência sutil à energia solar sem ser literal |
| **Espessura** | Traços grossos (bold) | Visibilidade em tamanhos pequenos. Não é delicado — é comandante |
| **Cantos** | Chanfrados (não totalmente arredondados) | Industrial precision. Referência: aviônica, dashboards de controle |

### 4.4 Paleta Cromática do Logo

| Papel | Cor | HEX | Significado |
|-------|-----|-----|-------------|
| **Âmbar Solar** (primária) | Dourado intenso | `#F5A524` | Energia, riqueza, sol no zênite |
| **Aurora Dourada** (gradiente) | Amarelo solar | `#FACC15` | Otimismo, luz, início de algo |
| **Flare Terminal** (gradiente) | Laranja profundo | `#E8740C` | Urgência, calor, combustão |
| **Fundo Comando** | Negro absoluto | `#09090B` | Autoridade, luxo, profundidade |
| **Mono Light** (inversão) | Branco quente | `#FAFAF9` | Para aplicação em fundos claros |

**Direção do gradiente:** De `#FACC15` (topo-esquerda) → `#E8740C` (baixo-direita).
Simula a trajetória da luz solar — do amanhecer ao zênite.

## 6. REGRAS DE "NÃO FAZER" (Logo Don'ts)

1. ❌ Nunca usar o logo sobre fundos que compitam com a paleta solar
2. ❌ Nunca rotacionar, distorcer ou achatar o logomark
3. ❌ Nunca aplicar sombras externas ou efeitos 3D
4. ❌ Nunca usar menos que 24px de tamanho mínimo
5. ❌ Nunca trocar as cores do gradiente
6. ❌ Nunca colocar texto sobre o logomark
7. ❌ Nunca usar sobre fundo branco puro sem inversão monocromática

## 7. BENCHMARKS VISUAIS DE REFERÊNCIA (NÃO COPIAR)

Marcas cujo **nível de qualidade e sofisticação** servem como norte (não como template):

| Marca | O que observar | Lição para Sunny |
|-------|---------------|-----------------|
| **Stripe** | Gradiente contido, geometria limpa | Gradientes podem ser premium quando controlados |
| **Linear** | Ícone geométrico, traços precisos | Simplicidade radical = memorabilidade |
| **Tesla** | T dentro de escudo → autoridade imediata | Container geométrico = selo de poder |
| **Vercel** | Triângulo único = identidade completa | Uma forma simples pode dizer tudo |
| **Coinbase** | Círculo + forma interna = moeda/valor | O container circular comunica completude |
| **Notion** | Preto e branco sem medo | Monocromático funciona quando a forma é forte |

## 8. PROMPTS PARA GERAÇÃO DA LOGO

> **Instruções:** Use os prompts abaixo em ferramentas de geração de imagem (Midjourney, DALL-E, Ideogram, etc.). Cada prompt é uma variação conceitual. Gere 4 resultados de cada e selecione os melhores para refinamento.

### PROMPT A — "O Raio no Selo" (Conceito Principal)

```
Minimalist vector logo design for a premium AI SaaS brand called "Sunny". 
The logomark features the letter "S" shaped as a sharp lightning bolt, 
centered inside a thin circular container with a subtle dashed stroke. 
A small arc at the bottom of the circle suggests a sunrise horizon line. 
The lightning-S has sharp geometric angles, bold weight, and a slight 15-degree 
forward lean suggesting speed and momentum. 

Color: warm amber-to-deep-orange gradient (#FACC15 to #E8740C) on the S-bolt. 
The circular container is a thin stroke in the same gradient. 
Background: pure black #09090B. 

Style: flat vector, no shadows, no 3D effects, no texture. 
Ultra-clean, geometric precision. Inspired by the visual quality of Stripe, 
Linear, and Tesla brand marks. 

The logo should work at 16px (favicon) and at billboard scale. 
No text, no wordmark — symbol only. 
Centered composition, square canvas, generous negative space around the mark.
```

---

### PROMPT B — "O Escudo de Comando" (Variação Hexagonal)

```
Minimalist vector logo design for a premium AI-powered solar sales platform 
called "Sunny". The logomark is a stylized letter "S" in the form of a 
lightning bolt, centered inside a hexagonal shield container. The hexagon 
has slightly rounded corners and a thin single-weight stroke. The S-bolt 
is bold and geometric with sharp angles.

A horizontal line cuts through the lower third of the hexagon, creating 
a subtle "horizon" effect — suggesting sunrise without being literal.

Color: linear gradient from golden yellow (#FACC15) at top-left to deep 
burnt orange (#E8740C) at bottom-right, applied to the S-bolt. 
The hexagonal container uses a 50% opacity version of the same gradient. 
Background: solid black #09090B.

Style: flat vector, ultra-minimal, zero decoration. No shadows, no glow, 
no 3D. Precision-engineered feel like aerospace or fintech branding. 
No text — logomark only. Square canvas, centered.
```

### PROMPT C — "A Moeda Solar" (Variação Premium/Luxury)

```
Minimalist vector logo design for a luxury-tier AI technology brand called 
"Sunny". The mark resembles a prestigious coin or medal: a bold, perfectly 
circular container with a double-line border (thin outer line, thicker inner line). 

Inside the circle, the letter "S" is rendered as a fluid, continuous stroke 
that evokes both a lightning bolt and a flowing energy wave. The S has uniform 
stroke weight, rounded terminals, and an elegant calligraphic quality while 
remaining strictly geometric.

At the very bottom of the inner circle, a thin semicircular arc suggests 
a rising sun on the horizon — the only solar reference, kept extremely subtle.

Color: monochromatic warm gold (#F5A524) on black (#09090B). 
No gradient — solid gold for maximum luxury perception. 
Style: engraved medal aesthetic, flat vector, no texture, no 3D.
Think: if Hermès designed a tech logo. Timeless, authoritative, unmistakable.
No text — symbol only. Square canvas, centered composition.
```

---

### PROMPT D — "O Bolt Puro" (Variação Ultra-Minimal)

```
Ultra-minimalist vector logo for a cutting-edge AI brand called "Sunny". 
The entire logo is a single geometric shape: the letter "S" constructed 
as a zigzag lightning bolt using exactly 6 straight lines and sharp 45-degree 
angles. No container, no circle, no additional elements.

The bolt-S stands alone with confident negative space around it. 
It has thick, uniform stroke weight giving it a monumental, architectural feel.
The top and bottom terminals of the S are cut at clean horizontal lines.

Color: solid warm amber (#F5A524) on pure black background (#09090B).
No gradient, no effects, no decoration. Maximum reduction.

Style: geometric brutalist minimalism. Inspired by Paul Rand's approach — 
reduce until you can't reduce any more. The mark should be drawable from 
memory after seeing it once.
No text, no wordmark. Symbol only. Square canvas, generous margins.
```

### PROMPT E — "Full Lockup" (Logo + Wordmark — SOMENTE após escolher o símbolo)

```
Professional brand lockup combining a [CHOSEN LOGOMARK] with the wordmark 
"Sunny" in Space Grotesk Bold typeface. The wordmark sits to the right 
of the logomark, vertically centered. 

The word "Sunny" uses tight letter-spacing (-2% tracking) and is rendered 
in warm white (#FAFAF9) against black background (#09090B). The logomark 
uses the brand gradient (#FACC15 to #E8740C).

Layout: horizontal lockup with 1.5x logomark-height spacing between 
symbol and text. The logomark height equals the cap-height of the wordmark.

Style: clean, corporate-premium, flat. No tagline, no additional elements. 
This is the primary brand signature for headers and formal use.
Square canvas with extra width, centered composition.
```

## 9. PROCESSO DE SELEÇÃO (Próximos Passos)

### Fase 1 — Geração
1. Gerar no mínimo **4 resultados por prompt** (A, B, C, D)
2. Total esperado: **16+ variações**

### Fase 2 — Triagem (Sagi Haviv 5-Pillar Test)
Avaliar cada resultado contra os 5 pilares:
- [ ] É **apropriado** para IA + Solar + B2B Premium?
- [ ] É **distinto** dos 9 concorrentes mapeados?
- [ ] É **simples** o suficiente para funcionar em 16px?
- [ ] É **memorável** após uma única exposição?
- [ ] É **atemporal** (funcionaria em 2016 e em 2036)?

### Fase 3 — Refinamento
1. Selecionar os **3 melhores** da triagem
2. Refinar vetorialmente (ajustar proporções, pesos, ângulos)
3. Testar em contextos reais: favicon, navbar, business card, dark/light

### Fase 4 — Lockup
1. Combinar o símbolo escolhido com o wordmark (Prompt E)
2. Produzir as **7 variações** listadas na Seção 5
3. Documentar no brandbook

### Fase 5 — Validação Final (Gut Feeling Test)
> "Se a logo não faz alguém sentir algo, recomeçamos." — Brand Chief

Perguntas de validação:
1. Ao ver esta logo pela primeira vez, você sente **autoridade**?
2. Ela parece pertencer a um **software de R$ 2.000/mês** ou a um de R$ 50/mês?
3. Você conseguiria **desenhá-la de memória** depois de vê-la uma vez?
4. Ela se destaca **completamente** dos concorrentes SolarZ, Aster, Azume?
5. Ela funciona em **preto e branco** sem perder identidade?

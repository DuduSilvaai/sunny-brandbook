# 🌞 Sunny Brandbook

> Brand identity system para **Sunny** — SaaS de inteligência artificial para energia solar.

## Sobre

Sunny é o nome da marca e do produto que transforma a operação comercial de integradoras de energia solar. O sistema combina CRM inteligente com SDR Agent (IA conversacional) para converter leads em reuniões qualificadas automaticamente.

**Filosofia:** Magnanimidade — "qualidade do que é grandioso, generoso e imponente"  
**Estilo visual:** Clean Luxury Tech / Elegant Dark Mode  
**Tipografia:** Space Grotesk  

## Estrutura

```
sunny-brandbook/
├── squads/brand/          # 7 agentes especialistas em branding
├── context/               # PRD, benchmarks, referências
│   ├── benchmarks/        # 9 concorrentes analisados
│   ├── aiox-reference/    # Referência do AIOX Brandbook (37+ páginas)
│   └── visual-reference.md
├── output/                # Outputs de cada fase
│   ├── 01-diagnosis/      # Fase 1: Brand Strategy
│   ├── 02-positioning/    # Fase 2: Competitive Context
│   ├── 03-naming/         # Fase 3: Verbal Identity
│   ├── 04-identity/       # Fase 4: Visual System
│   ├── 05-activation/     # Fase 5: Movement & Launch
│   └── brandbook-site/    # Site final do brandbook
└── README.md
```
